# 智能彩票分析检测系统

一个基于 Streamlit 的智能彩票投注分析检测系统，支持多种彩票类型的异常投注行为检测。

## 🎯 功能特点

- **多彩种支持**: PK拾/赛车、时时彩、六合彩、快三、三色彩
- **智能检测**: 超码检测、矛盾投注、异常模式识别
- **数据清洗**: 自动识别列名、数据质量验证
- **可视化界面**: 现代化的 Streamlit Web 界面
- **结果导出**: 支持 Excel 格式导出

## 🚀 快速开始

### 本地运行

pip install -r requirements.txt
streamlit run app.py
